# CMS Frontend Development Skills

You are building a professional CMS dashboard for a luxury safari website.

Key principles:

Keep UI simple
Avoid complex forms
Prioritize content editing

UX rules:

• Sidebar navigation
• Large form inputs
• Image previews
• Save buttons clearly visible

Do NOT allow editing of layout or styling.

Editable fields must focus on:

content
images
SEO fields

UI components to create:

Sidebar
Dashboard cards
Data tables
Content editor forms
Image uploader
Gallery selector

Forms must include:

save button
loading state
success notification
error message

Performance:

CMS must be lazy-loaded under /cms.

Public website bundle must not include CMS code.
